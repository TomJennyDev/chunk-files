## 📦 What's Included

This documentation covers a complete file processing system built with:

- **NestJS API** - File upload and search service
- **LocalStack** - AWS emulation (S3, SQS, Lambda)
- **Lambda Workers** - Serverless file processing
- **Elasticsearch** - Distributed search engine
- **Kibana** - Data visualization and monitoring
